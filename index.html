X.E., Kakapo team

Приложение для поиска потерянных вещей Apptofind использует геолокацию для поиска потерянных вещей. Маркеры на карте - отметки потерянных и найденных вещей. Вы можете легко просмотреть нужные вам територии, откликнуться на объявления и разместить свое. Удобный поиск позволяет быстро обнаружить пропажу, а личный кабинет не упустить никаких деталей. Загрузите наше приложение для Android c API от 21 уровня.
